# 📖 Siren 机器人使用手册

欢迎使用 **Siren**！本机器人通过监控视频更新并提供强大的查询指令，让你在 Discord 中也能掌握第一手资讯。

## 🚀 核心功能

- **实时监控**: 自动检查 MissAV 最新视频，发现新片后立即推送到指定频道。
- **美观展示**: 采用 Embed 卡片，包含封面、番号、演员、时长等完整信息。
- **完全异步**: 基于 Python 异步框架，响应极速，资源占用低。

## 🎮 指令指南

所有指令均为 **斜杠指令 (Slash Commands)**，只需在对话框输入 `/` 即可看到提示。

### 1. 基础查询
- `/help`: 显示详细的帮助菜单。
- `/status`: 查看机器人当前运行状态（运行时间、数据库视频总数、检测频率等）。
- `/latest [count]`: 查看视频库中最新的 `count` 个视频（默认 5 个）。

### 2. 交互式搜索
- `/search [keyword]`: 根据关键词（标题、演员、番号）搜索视频库及在线内容。
- `/info [code]`: 输入完整番号，直接获取该视频的详细信息（带预览）。

### 3. 订阅管理
- `/subscribe`: 订阅当前频道。开启后，每当有新视频产生，机器人会自动在该频道 @everyone。
- `/subscribe_actress [name]`: 针对性订阅特定演员。该演员有新作时会给予提醒。
- `/subscribe_tag [tag]`: 订阅特定标签（如“巨乳”、“精选”等）。
- `/list`: 查看当前服务器/频道的详细订阅清单。
- `/unsubscribe`: 取消当前所有的订阅。

### 4. 高级功能 (需管理员权限)
- `/crawl_code [code]`: 强制爬取特定番号的信息并存入库中。
- `/crawl_actor [name] [limit]`: 手动触发对某位演员作品的深度挖掘（限制前 `limit` 部）。

## 💡 使用小贴士

- **同步指令**: 在 Discord 中添加机器人后，可能需要几分钟同步斜杠指令列表，或者尝试重启 Discord 客户端。
- **卡片交互**: 通知消息中包含磁力链接（若解析成功）或详情页跳转，方便一键直达。
- **演员订阅**: 建议使用演员的官方正式名称以获得最高准确率。
